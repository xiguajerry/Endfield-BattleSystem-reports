    伤害与攻击¶

    Damage — 伤害¶

    C#: Beyond.Gameplay.Core.DamageAction+DamageActionData
    字段 类型 说明
    alwaysNext bool 始终执行下一个 Action
    attacker ActionTargetType 攻击方
    targetSettings TargetSettings 目标设置
    effectSource TargetSettings 效果来源 (用于播放受击特效的实体)
    damageUnits List\<DamageUnit> 伤害段列表 (支持多段伤害)
    hitEnvironment bool 是否启用环境打击
    hitEnvData HitEnvData 环境打击数据

    每段 DamageUnit 包含伤害公式、属性类型、伤害修饰器等。支持 _CalculateDamageResultByType 分类型计算 (普通实体/独立生命/塔)。

    SetAllowedDamageDecoMask — 设置允许伤害装饰 Mask¶

    C#: Beyond.Gameplay.Core.SetAllowedDamageDecoMask+Data
    字段 类型 说明
    target TargetSettings 设置目标
    mask DamageDecorateMask 动作期间允许通过的伤害装饰 mask

    CharAlertJump — 角色警戒跳跃¶

    C#: Beyond.Gameplay.Core.CharAlertJumpAction+CharAlertJumpActionData

    角色进入警戒状态时的跳跃行为 (用于敌人AI)。

    • SwitchAction：按 choice 解析黑板值，匹配 options[].value 后执行对应 actionData。alwaysNext=false 时，如果没有命中 option，或命中的子序列返回失败，会停止父序列后续 action；alwaysNext=true 时保持继续执行。
    • ForEachAction：按 target 解析目标列表，逐个把 targetEntity 切换为当前目标并执行子 action，用于 target group 多目标分发。
    • RandomAction：按 randomType 在 minValue/maxValue 范围内生成 Int 或 Float，并写入 targetBlackboardKey；支持 EntityBB_ 动态路由。

    时停与时间控制¶

    TimeDilation — 时间膨胀/时停¶

    C#: Beyond.Gameplay.Core.TimeDilationAction+Data
    字段 类型 说明
    layer TimeDilationTarget 影响层级 (Attacker/Target/Both/Global)
    slot GameplayTag 时间槽位标签
    timeDilationPriority GameplayTag 时停优先级 (高优先级覆盖低)
    duration BlackboardDouble 持续时间
    useCurveKey bool 使用曲线键
    curveKey string 曲线键 (从 CurveConfig 读取)
    timeScaleCurve AnimationCurve 时间缩放曲线 (直接定义)
    finishByAction bool 被后续 Action 结束
    ignoreTargets List\<TargetSettings> 忽略的目标
    effectTargets List\<TargetSettings> 需要播放效果的目标
    useTimeScaleForSkillCdTick bool 是否影响技能CD计时
    influenceSkillCdTime BlackboardDouble 影响的技能CD时间

    HitStop — 打击停顿¶

    C#: Beyond.Gameplay.Core.HitStopAction+Data
    字段 类型 说明
    affectType AffectType 影响类型 (OnlyAttacker/OnlyTarget/Both)
    curveKey string 时间曲线键
    useDirectCurve bool 使用直接曲线
    directCurve AnimationCurve 直接定义的曲线
    duration float 停顿持续时间
    timeDilationPriority GameplayTag 停顿优先级
    attacker TargetSettings 攻击方 (停顿方)
    target TargetSettings 目标方 (停顿方)

    UltimateTime — 大招时停¶

    C#: Beyond.Gameplay.Core.UltimateTimeAction+Data

    释放大招时的全局时停效果 (通常比普通 HitStop 更长，影响范围更广)。

    PauseComboSkillTime — 暂停连招技能时间¶

    C#: Beyond.Gameplay.Core.PauseComboSkillTime+Data

    在连招切换期间暂停技能时间窗口。

    SetIgnoreGlobalTimeScale — 设置忽略全局时间缩放¶

    C#: Beyond.Gameplay.Core.SetIgnoreGlobalTimeScaleAction+Data

    让特定实体忽略全局时间缩放 (不被打断/时停影响)。

    TickInterval — Tick间隔¶

    C#: Beyond.Gameplay.Core.TickIntervalAction+Data

    控制 Action 序列的 tick 间隔 (类似 WaitAction)。
