    Skill Saved Target¶

    Fields¶

    Field Offset Notes
    _targetHandle 0xC0 Mutable TargetHandle used by the current skill.

    Methods¶

    Method 状态 Status Notes
    StoreSmartTarget(TargetHandleView defaultTargetView, TargetHandleView comboSkillTargetView, TargetHandleView comboSkillTriggerView, string contextKey) 已确认 已确认 Builds SmartTargetSelectSetting, calls SelectorUtils.FindSmartTarget, falls back to GetDefaultTarget, then updates action context target group.

    SaveTarget:

    1. Loads this + 0xC0 as _targetHandle; null is a hard error path.
    2. Calls TargetHandle.Clear(clearEntity=true, clearPosition=true, clearSlot=true).
    3. If input TargetHandleView.m_targetHandle is null, exits.
    4. Otherwise, checks target.isValid.
    5. If not valid but hitReactionInfos.Count > 0, copies hitReactionInfos[0].

    StoreSmartTarget:

    1. Reads SkillData.smartTargetSelectStrategy, smartTargetBuffIds, smartTargetTagQuery and smartTargetBuffFindSettings into a stack SmartTargetSelectSetting.
    2. Uses Skill.get_castDistance() when defaultTargetView exists, otherwise SkillSetting +0x7C as select range.
    3. Reads owner faction through Skill.get_owner().get_factionIndex().
    4. Calls SelectorUtils.FindSmartTarget(setting, faction, selectRange, defaultTargetView, comboSkillTargetView, comboSkillTriggerView).
    5. If result is null, calls Skill.GetDefaultTarget(limitedMainTargetDistance).
    6. Calls IActionEnvironment.Context.UpdateTargetGroup(contextKey, targetHandle).

    Lifecycle¶

    • AbilitySystem.TryCastSkill calls Skill.SaveTarget(inputTarget, options) before BeforeCastStart and Skill.DoCast.

    • GetDefaultTarget spatial scoring, range and screen checks are not simulated.
