    chr_0029 骏卫 (Pograni) 终结技¶

    概述¶

    多段式物理终结技，消耗 100 USP。骏卫展开护盾冲向目标并造成大范围牵引+击飞，随后进行 4 次瞬移追击，最后以大范围终结一击收尾。可通过 combo skill 在前一次释放后连段继续攻击。全程霸体+无敌。

    基础数据¶

    字段 值
    skillId chr_0029_pograni_ultimate_skill
    castType Active
    skillSpecification CharacterUltimateSkill
    attackRangeType Melee (近战)
    damageType Physical (物理)
    durationFrame 210 (约 3.5s)
    exclusiveFrame 90 (约 1.5s)
    cooldownTime 0.0
    cost (UltimateSp) 100.0

    技能结构¶

    终结技由三层 skill 组成:

    flowchart TD
    Main["主技能 chr_0029_pograni_ultimate_skill<br/>210f"]
    Spawn["frame 74<br/>生成 4 个 AbilityEntity"]
    RushHit["frame 76<br/>突进伤害"]
    Combo["后续由 combo 触发追击与终结"]

    AE["AbilityEntity skill<br/>300f"]
    Move["1-7f<br/>SoldierRush 冲向目标"]
    BuffRace["3-17f<br/>竞争创建主 buff"]
    Finish["50f<br/>AbilityEntity 自毁"]

    Main --> Spawn --> AE
    Main --> RushHit --> Combo
    AE --> Move --> BuffRace --> Finish

    子 skill:

    • chr_0029_pograni_ultimate_skill_abilityentity_attack2 — 追击阶段 (400f, 4 轮)
    • chr_0029_pograni_ultimate_skill_abilityentity_finish1~4 — 终结阶段

    伤害窗口¶

    突进 (Rush) — 主技能 frame 76-82¶

    对 centerRange 组（以目标为中心, center_radius=6.0 球形检测的敌方）造成:

    参数 值
    攻击系数 atkScale = atk_scale_rush = 4.0 (400% ATK)
    伤害类型 Physical / HP
    damageDecorateMask 4608
    simpleCalculation true
    失衡参数 值
    ------ -----
    失衡伤害 poise_rush = 15 (DefiniteValueCalculation)
    伤害类型 Physical / Poise

    附带效果:

    • PullAction: 牵引 centerRange 内敌人到 mainPos（目标反方向 180°），pullSpeed=30，持续 0.167s，受 superArmor 衰减 (20→0.75, 30→0.5, 35→0)
    • BlowOffAction: 击飞高度 1.0
    • InterruptAction: 定身 1.0s
    • EnemyHurtAnimAction: 播放受击动画

    注意: 突进伤害 不产生 ATB (gainCost: false)

    追击 (Trigger) — attack2 ×4 轮¶

    每轮在 teleport 到目标附近后:

    参数 值
    攻击系数 atkScale = atk_scale_trigger = 2.0 (200% ATK)
    damageDecorateMask 512
    simpleCalculation true
    4 轮时间 frame 9/108/209/307
    ATB 获取 ObtainCost(Atb, Gain, atb_trigger=15.0) per hit

    终结一击 (Finish) — finish4¶

    参数 值
    攻击系数 atkScale = atk_scale_final = 4.0 (400% ATK)
    damageDecorateMask 4608
    失衡伤害 poise_final = 15 (DefiniteValueCalculation)
    ATB 获取 ObtainCost(Atb, Gain, atb_final=50.0)

    附带效果:

    • TakeDownAction: 击倒
    • HitStopAction: 卡帧
    • EnemyHurtAnimAction: 受击动画

    终结一击通过 CheckTimedMarkerCondition 条件触发。条件满足时执行完整终结，不满足只执行 HitStop。

    总倍率¶

    段 倍率 ATB
    突进 Rush 400% + 15 poise 0
    追击 ×4 200% × 4 = 800% 15 × 4 = 60
    终结 Final 400% + 15 poise 50
    合计 1600% ATK + 30 poise 110 ATB

    注: finish1-3 的伤害为备选终结路径，系数来自 blackboard atk_scale（需在子 skill blackboard 中确认），ATB 来自 atb key。

    自卫机制¶

    效果 时间 说明
    无敌 (DamageImmune) frame 0-90 buff_common_damage_immune_medium
    霸体 (SuperArmor) frame 0-90 SetSuperArmorAction
    时缓 frame 0-3 TimeDilation (HitStop 层)
    隐藏 UI frame 0-75 HideUIAction
    CG 特写 frame 0-75 UltimateTimeAction + UltimateShowAction

    ATB 获取分析¶

    来源 方法 值 条件
    追击 attack2 ×4 Gain 15.0 × 4 命中
    终结 finish4 Gain 50.0 TimedMarker 条件满足
    突进 rush — 0 不产生 ATB

    关键: 所有 ATB 获取均使用 Gain 方法，与战技的 Return 不同。这意味着:

    • 终结技的 ATB 获取 会触发天赋1 OnObtainAtb 事件（因为 CheckObtainAtbType 匹配 method=Gain）
    • 终结技是骏卫叠天赋层数的主要手段之一

    damageDecorateMask 对比¶

    技能 Mask 说明
    战技 (normal) 4352 NormalSkill 标记
    突进/终结 (rush/final) 4608 UltimateSkill 标记

    与天赋1的互动¶

    终结技的所有 ATB 获取均使用 Gain 方法，会触发 OnObtainAtb 事件，可叠加天赋1层数。追击每轮 +15 ATB × 4 = 60，终结 +50 ATB = 110，合计单次终结技可获取 110 ATB。
