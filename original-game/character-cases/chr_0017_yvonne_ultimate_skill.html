    chr_0017_yvonne Yvonne 终结技流程分析¶

    更新时间: 2026-06-01

    时间轴总览¶

    timeline
    title chr_0017_yvonne_ultimate_skill
    0f : 创建主 buff
       : 写入 potential4/5 参数
    3f : UltimateShow
       : UltimateTime / HideUI / 镜头 / 特效
    57-64f : 生成 3 个 AbilityEntity
    61-80f : 清理旧 buff / 时停 / 无敌 / 霸体 / 武器显示
    126f : 主 skill 结束

    主 buff 生效 12 秒，期间切换到 ult mode 并改写普攻链；主 buff 结束后创建 end buff，提供终结攻击入口。

    阶段 关键行为
    OnBuffStart 创建环境 buff，并按 potential 条件创建 potential4/5 相关 buff
    DuringBuffEnable SwitchModeAction(modeId=ult)；RefrainObtainUsp
    ult 普攻链 Attack 入口切到 chr_0017_yvonne_ult_attack1_1，后续进入 ult 连段
    OnBuffFinish 创建 buff_chr_0017_yvonne_ultimate_skill_end
    end buff Attack 入口切到终结攻击 chr_0017_yvonne_ult_attack_end

    结论¶

    Yvonne 大招不是单段爆发，而是一个 12 秒的模式切换技能。 核心结构是：

    flowchart TD
    Cast["主 UltimateSkill"]
    Buff["创建主 buff<br/>duration=12s"]
    UltMode["DuringBuffEnable<br/>切换 mode=ult"]
    Attack{"玩家触发 Attack"}
    Chain["ult 普攻链<br/>projectile 多段水晶伤害"]
    Active{"主 buff 仍存在"}
    Finish["OnBuffFinish"]
    EndBuff["创建 end buff<br/>duration=3s"]
    EndAttack{"end buff 期间触发 Attack"}
    EndHit["ult_attack_end<br/>终结水晶伤害"]

    Cast --> Buff --> UltMode --> Attack
    Attack --> Chain --> Active
    Active -->|是| Attack
    Active -->|否| Finish --> EndBuff --> EndAttack
    EndAttack -->|触发| EndHit
    EndAttack -->|未触发| EndBuff

    主 buff¶

    buff_chr_0017_yvonne_ultimate_skill:

    字段 值
    duration 使用黑板 duration，默认 12

    事件：

    事件 行为
    OnBuffStart 创建环境 buff；按 potential 条件创建 potential4/5 相关 buff
    DuringBuffEnable SwitchModeAction(modeId=ult)；RefrainObtainUsp
    OnBuffFinish 创建 buff_chr_0017_yvonne_ultimate_skill_end
    OnOwnerHpZero 给已召唤的 UltimateAbilityEntity 创建 robot_end buff

    这里最关键的是 SwitchModeAction(modeId=ult)。Yvonne 大招的持续输出不是主大招 timeline 自己打出，而是 mode 切换后重写普攻入口。

    projectile hit 伤害¶

    chr_0017_yvonne_ult_attack1_projhit 与 chr_0017_yvonne_ult_attack2_projhit 都是 0f hit skill：

    字段 值
    damageType Cryst
    damageAttributeType Hp
    atkScale 使用黑板 atk_scale，默认值 0.9
    simpleCalculation true

    由于发射动作会给 projectile 写入黑板，实际 atk_scale 由上游 attack skill 控制；如果没有写入，默认 0.9。

    end buff 与终结攻击¶

    主 buff 结束时创建 buff_chr_0017_yvonne_ultimate_skill_end。

    字段 值
    duration 使用黑板 duration_end，默认 3

    事件：

    事件 行为
    DuringBuffEnable ComboCacheAction，把 Attack 映射到 chr_0017_yvonne_ult_attack_end；继续 RefrainObtainUsp
    OnBuffStart potential5 相关处理；清理部分旧状态
    OnBuffFinish 给 AbilityEntity 创建 robot_end，结束 shield/environment/potential/full_effect buff，立刻设置 UltimateSkill CD

    chr_0017_yvonne_ult_attack_end:

    字段 值
    durationFrame 161f
    attackRangeType Ranged
    castDistance 12

    关键 timeline：

    帧 行为
    0-3f FinishBuffAdvanced，清理前置状态
    7-8f / 12-13f / 17-18f 给已召唤 UltimateAbilityEntity 创建 robot_end buff
    28-60f hitbox 找目标，结算终结伤害
    28-31f 结束 ultimate_skill_end 和 layer buff
    67-70f 结束 camera buff

    终结伤害：

    单元 类型 说明
    HP damage Cryst 使用黑板 atk_scale，默认 1.35
    Poise damage Cryst / Poise 仅主控启用
    Extra damage Cryst 条件分支使用黑板 atk_scale_extra = 4.0

    终结攻击会读取 buff_chr_0017_yvonne_ultimate_skill_layer 的 stack，并写入动态黑板：

    normal_dmg_up
crit_rate_up

    这些动态值会作为 DamageProcessor 参数影响终结段伤害。 要完整模拟 Yvonne 大招，至少需要以下能力：

    1. SwitchModeAction(modeId=ult / ult_end) 改写运行时 Attack 映射和 curNormalAttackList。
    2. ComboCacheAction / AllowNextSkillAction 驱动 ult 普攻链。
    3. LaunchProjectile 创建 projectile hit skill，并传递 projectile 黑板。
    4. DamageAction 支持 Cryst 伤害、poise damage 和 damage processor。
    5. OnBuffFinish 事件必须可驱动 ultimate_skill_end 创建和清理。
    6. GetTargetBuffBBAdvanced / SaveBuffStackNumAdvanced / ModifyDynamicBlackboard 需要支持终结段 layer stack 加成。
    7. AbilityEntity 不只是视觉对象：至少要能按 tag 找到 OwnerSpawnedEntity，并在 end 阶段接收 robot_end buff。
