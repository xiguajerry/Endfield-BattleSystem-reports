    TimeDilationConfig¶

    本页合并多个源文档以去除同名 TimeDilation/Tick 类页的重复导航，同时保留每个源文件的原始信息。

    TimeDilationConfig¶

    字段¶

    字段 Offset 说明
    priorityMap: SerializeFieldDictionary<GameplayTag, int> 0x18 TimeDilation priority tag 到整数优先级的映射。TimeDilationManager._AddNew* 用该值做同 slot 竞争。
    curveMap: SerializeFieldDictionary<string, AnimationCurve> 0x20 action curveKey 到 Unity AnimationCurve 的映射。instance OnTick 每帧按 normalized time 评估曲线。
    slotSpecialConfig: List<SlotSpecialConfig> 0x28 global slot 与 entity slot 的特殊关系配置；当前数据中 Global/UltiSkill 与 Global/GamePlay 都会影响 Entity/Frozen duration。
    字段 Offset 说明
    globalSlot: GameplayTag 0x10 Global TimeDilation slot。
    entitySlot: GameplayTag 0x20 Entity TimeDilation slot。
    isInfluenceDuration: bool 0x30 当 global slot 生效时，该 entity slot 的 duration 是否按 global time scale 推进。

    方法¶

    方法 确认状态 已确认行为
    DataManager.get_timeDilationConfig() 已确认 返回 DataManager 持有的 TimeDilationConfig。
    DataManager.set_timeDilationConfig(TimeDilationConfig) 已确认 设置 DataManager 持有的 TimeDilationConfig。
    TimeDilationConfig.ctor 已确认 ScriptableObject 配置容器构造；无 gameplay 行为。
    SlotSpecialConfig.ctor 已确认 嵌套配置项构造；无 gameplay 行为。

    调用入口¶

    • TimeDilationManager._AddNewEntityTimeDilationInst / _AddNewGlobalTimeDilationInst 读取 priorityMap 判定同 slot 竞争。
    • TimeDilationAction.ExecuteInternal 传入 curveKey 或 direct curve；manager/instance 使用 curveMap 或 action curve 构造实例曲线。HitStopAction 的 keyed curve owner 是 HitStopConfig，不是本类。
    • EntityTimeDilationInst.OnTick 根据 slotSpecialConfig 判断 Entity/Frozen duration 是否受 active global TimeDilation 影响。

    生命周期¶

    1. DataManager 加载 TimeDilationConfig asset。
    2. Battle / action runtime 启动 TimeDilation 时读取 priority、slot special config、curve。
    3. TimeDilation instance lifetime 内每帧评估曲线。
    4. 配置对象自身不参与 tick。

    已确认行为¶

    • priorityMap 当前 mined data 中包含：
    • TimeDilation/Priority/UltiSkill = 100
    • TimeDilation/Priority/Frozen = 50
    • TimeDilation/Priority/ComboSkill = 30
    • TimeDilation/Priority/GlobalSlowMotionPro = 21
    • TimeDilation/Priority/BreakPoise = 20
    • TimeDilation/Priority/Interrupt = 15
    • TimeDilation/Priority/HitStop = 10
    • TimeDilation/Priority/DashSucceed = 10
    • TimeDilation/Priority/GlobalSlowMotion = 10
    • TimeDilation/Priority/VisualAdjust = 50
    • curveMap 当前 mined data 包含 forge_iron_hitstop、RESETto1、ComboSkill 等 key，值为 Unity keyframe array；HitStopAction keyed curve 的正常来源仍是 HitStopConfig。
    • forge_iron_hitstop 曲线首帧 value 为 0.05，末帧 value 为 1.0。
    • RESETto1 曲线全程 value 为 1.0。
    • slotSpecialConfig 当前两项：
    • Global/UltiSkill -> Entity/Frozen，isInfluenceDuration=true
    • Global/GamePlay -> Entity/Frozen，isInfluenceDuration=true

    TimeDilationConfig¶

    数据源¶

    TimeDilationConfig 是 TimeDilation 系统的数据配置，至少包含 priority、curve、slot special config 和 predefined tags。

    priorityMap¶

    Priority Tag Score
    TimeDilation/Priority/UltiSkill 100
    TimeDilation/Priority/Frozen 50
    TimeDilation/Priority/VisualAdjust 50
    TimeDilation/Priority/ComboSkill 30
    TimeDilation/Priority/GlobalSlowMotionPro 21
    TimeDilation/Priority/BreakPoise 20
    TimeDilation/Priority/Interrupt 15
    TimeDilation/Priority/HitStop 10
    TimeDilation/Priority/DashSucceed 10
    TimeDilation/Priority/GlobalSlowMotion 10

    slot tags¶

    Slot 用途
    TimeDilation/Layer/Global/GamePlay Global TimeDilationAction 固定 slot
    TimeDilation/Layer/Global/UltiSkill UltimateTime slot
    TimeDilation/Layer/Entity/Frozen Entity 冻结 / slotSpecialConfig 映射目标
    TimeDilation/Layer/Entity/VisualAdjust 视觉调节类 Entity scale

    curveMap¶

    forge_iron_hitstop

    影响：

    • 空曲线按 Unity AnimationCurve.Evaluate() 行为应为 0.0。

    slotSpecialConfig¶

    当前配置中存在两个映射：

    [
 {
 "globalSlot": "TimeDilation/Layer/Global/UltiSkill",
 "entitySlot": "TimeDilation/Layer/Entity/Frozen",
 "isInfluenceDuration": true
 },
 {
 "globalSlot": "TimeDilation/Layer/Global/GamePlay",
 "entitySlot": "TimeDilation/Layer/Entity/Frozen",
 "isInfluenceDuration": true
 }
]
