    TickOwnerInfo¶

    本页合并多个源文档以去除同名 TimeDilation/Tick 类页的重复导航，同时保留每个源文件的原始信息。

    TickOwnerInfo¶

    状态：documented（已确认 for construct/register/scale methods） 更新时间：2026-06-11

    原游戏来源¶

    • 原游戏类：TickOwnerInfo

    字段¶

    字段 类型 说明
    m_owner ITickOwner tick owner。
    m_useUnscaledDeltaTime bool 是否使用 unscaled delta。
    m_ignoreGlobalTimeScale bool owner 是否忽略所属 group/global scale。
    m_groupType TickOwnerInfoGroupType 所属 group 类型。
    m_additionalTimeScale float owner 额外 scale，construct 默认 1.0。
    m_forceTick bool 是否强制 tick。
    m_supportOnTimeScaleChange bool owner 是否支持 time-scale callback。
    m_tickOwnerInfoGroup TickOwnerInfoGroup 所属 group。
    m_isConstructed / m_isRegistered bool 生命周期状态。
    m_tickFunctions List<TickFunction> owner 下 tick functions。
    m_hasActiveTickFunction bool 是否存在 active tick function。

    调用入口¶

    TimeManager 注册 ITickOwner 时创建/读取 TickOwnerInfo，调用 Construct 和 OnRegister。OnRegister 通过 TimeManager 获取 TickOwnerInfoGroup 并加入；OnUnregister 从 group 移除。

    生命周期¶

    1. Construct：
    2. 写 owner
    3. m_isConstructed = true
    4. 若 owner 存在，调用 owner 的 support query，写 m_supportOnTimeScaleChange
    5. 写 m_ignoreGlobalTimeScale
    6. m_additionalTimeScale = 1.0
    7. 写 m_groupType
    8. OnRegister：
    9. 根据 groupType 和 ignoreGlobal 获取 group
    10. 加入 group
    11. m_isRegistered = true
    12. SetIgnoreGlobalTimeScale：
    13. 如果未变化，直接返回
    14. groupType 为 1 时不迁移 group
    15. 已注册时从旧 group 移除，再加入新 group
    16. OnUnregister：
    17. 从当前 group 移除
    18. 调用 Reset
    19. m_isRegistered = false

    TickOwnerInfo¶

    类型职责¶

    TickOwnerInfo 表示一个 tick owner 的 timeScale 状态。它把 owner 自身的 additional scale 与 group final scale 相乘，得到这个 owner 实际使用的 scale。

    GetTimeScale¶

    if m_useUnscaledDeltaTime:
 groupScale = 1.0
else:
 groupScale = m_tickOwnerInfoGroup.m_timeScale
return m_additionalTimeScale * groupScale

    关键结论¶

    • ignore global / unscaled owner 不吃 group scale。
    • owner 自身 additional scale 仍会参与最终结果。
    • TimeDilationManager 能在全局时停中继续 tick，正是因为它作为 tick manager 使用 ignore global scale。
