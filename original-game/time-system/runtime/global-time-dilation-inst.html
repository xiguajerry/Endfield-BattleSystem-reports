    GlobalTimeDilationInst¶

    本页合并多个源文档以去除同名 TimeDilation/Tick 类页的重复导航，同时保留每个源文件的原始信息。

    GlobalTimeDilationInst¶

    字段¶

    字段 Offset 说明
    m_duration: float 0x18 持续时间，单位秒；小于 0 表示无限期。
    m_ignoreTargetList: List<ObjectPtr<Entity>> 0x20 忽略 global scale 的目标列表。
    m_scaleCurve: AnimationCurve 0x28 scale 曲线。
    m_constScale: float? 0x30 常量 scale；存在时不 evaluate curve。
    m_elapsedTime: float 0x38 已经过时间。
    m_slotTag: GameplayTag 0x40 Global slot。
    m_priorityTag: GameplayTag 0x50 priority tag。
    m_useTimescaleForSkillCdTick: float? 0x60 技能 CD 受影响的时间窗。
    m_timeScaleId: int? 0x68 TimeManager.StartChangeTimeScale 返回的 key。

    方法¶

    方法 确认状态 已确认行为
    ctor 已确认 初始化 ignore target list 等字段。
    Reset(List<Entity>, float, float?, GameplayTag, GameplayTag, AnimationCurve, float?) 已确认 清 elapsed/handle，写 duration/curve/constScale/slot/priority/CD influence window，并复制 ignore target list。
    Start() 已确认 对 ignore target list 调 Entity.SetIgnoreGlobalTimeScale(true)，准备 global scale 生效。
    OnTick(float) 已确认 constScale 存在时使用 constScale，否则按 elapsed/duration evaluate curve；首次调用 TimeManager.StartChangeTimeScale(scale, TimeDilation, instId)，后续 ChangeTimeScaleByExistingKey；最后 elapsed += deltaTime。
    Stop() 已确认 停止 TimeManager key，并对 ignore target list 恢复 SetIgnoreGlobalTimeScale(false)。
    get_influenceSkillCd() 已确认 m_useTimescaleForSkillCdTick.HasValue && m_useTimescaleForSkillCdTick.Value >= m_elapsedTime。
    get_isValid() 已确认 m_timeScaleId.HasValue，且 duration + epsilon >= elapsed 或 duration < 0。

    调用入口¶

    • Global 层 TimeDilationAction。
    • UltimateTimeAction / StartUltimateTime。

    生命周期¶

    1. Reset 清理旧状态并记录 ignore target。
    2. manager add 成功后调用 Start()。
    3. 每帧 OnTick 创建或更新 TimeManager scale source。
    4. Stop 清理 TimeManager source 与 ignore target flag。
    5. isValid == false 后由 manager 移除/回收。

    已确认行为¶

    • Global instance 通过通用 TimeManager 影响 tick groups。
    • TimeManager active timeScale 由 TimeManager 选择，不由 Global instance 自行合并。
    • 技能 CD influence 只看 nullable window 和 elapsed；没有 window 时不影响 CD。

    OnTick 行为¶

    if constScale.HasValue:
 timeScale = constScale.Value
else:
 normalizedTime = elapsed / duration
 transformed = time transform(normalizedTime)
 timeScale = scaleCurve.Evaluate(transformed)
timeScale = max(0.0, timeScale)
apply to TimeManager
elapsed >= duration -> expire

    Global instance 影响全局 TimeManager，但 ignore target list 中的 entity 不应吃到这个 global scale。

    influenceSkillCd¶

    useTimescaleForSkillCdTick.HasValue &&
useTimescaleForSkillCdTick.Value >= elapsedTime

    含义：

    • 只有 Global instance 可以影响技能 CD tick。
    • 必须显式传入 nullable influence window。
    • window 过后，即使 Global TimeDilation 还在，技能 CD 也恢复正常 tick。
    • Entity HitStop / Frozen 不默认暂停技能 CD。
